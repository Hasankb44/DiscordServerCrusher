# DiscordServerCrusher

## Properties

- `!nuke` crush server

## setup

1. Copy this repo:
    ```bash
    git clone https://github.com/Hasankb44/DiscordServerCrusher.git
    cd DiscordServerCrusher
    ```

2. Load Dependencies:
    ```bash
    mvn clean install
    ```

3. Set your token on src/java/main/org/bot/DiscordBot.java

4. Start the bot:
    ```bash
    mvn compile
    ```
    veya:
    ```bash
    java -cp target/classes/org/Main
    ```
